⚡ Demo Vite

Projeto demonstrativo das principais funcionalidades do Vite

🔥 Hot Module Replacement (HMR)

Edite src/main.js ou src/style.css com o servidor de dev rodando — o navegador atualiza sem perder estado.

📦 Build & Code Splitting

Rode npm run build para gerar o bundle em dist/. Vite usa Rollup em produção, com tree-shaking, minificação e hash nos nomes dos arquivos. 


        


        

          
🌱 Variáveis de Ambiente

          

            Vite expõe variáveis prefixadas com VITE_ via
            import.meta.env.
          

          

        


        

          
🧪 Testes com Vitest

          

            Rode npm test para executar a suíte. O Vitest
            compartilha a configuração do Vite — sem duplicação.
          

        


        

          
📊 Análise de Bundle

          

            Rode npm run build:analyze para gerar
            dist/stats.html com a visualização do bundle.
